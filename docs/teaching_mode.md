# Teaching Mode

Teaching mode generates slides optimized for pacing and understanding.

**Command**
```bash
researchos -a 2401.12345 --teaching-mode -s 12 -b 4
```
## Workflow
![Workflow](../images/teaching_mode_workflow.png)


**Behavior**
- More intuition
- More examples
- Fewer equations
- Adds “Pause Question” slides for reflection

**Best For**
- TA sessions
- workshops
- revision lectures
